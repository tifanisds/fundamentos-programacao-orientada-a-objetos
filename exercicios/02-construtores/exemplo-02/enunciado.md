# Exercício 2 — Classe Produto com Construtor

Crie uma classe chamada Produto.

### A classe deve possuir os seguintes atributos:

- `nome`

- `preco`

- `quantidade`

### Crie um construtor que receba esses três valores.

O construtor deve inicializar todos os atributos.

### Crie dois métodos:

`mostrarProduto()`

Exibe os dados do produto:
```java
Produto: Mouse
Preço: 80.0
Quantidade: 10
```

`valorTotalEstoque()`

Esse método deve calcular:
 
`preco * quantidade`

e mostrar:

`Valor total em estoque: 800.0`

### No método `main`:

1 - Crie um objeto Produto usando o construtor

2 - Chame os métodos mostrarProduto() e valorTotalEstoque()
