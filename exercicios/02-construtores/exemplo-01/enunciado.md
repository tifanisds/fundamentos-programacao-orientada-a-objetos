# Exercício 1 — Classe Pessoa com Construtor
### Crie uma classe chamada Pessoa.

A classe deve possuir os seguintes atributos:

- `nome`

- `idade`

### Crie um construtor que receba:

- `nome`

- `idade`

Esse construtor deve inicializar os atributos da classe.

### Crie também um método:
`apresentar()`

Esse método deve exibir:
```java
Nome: João
Idade: 28
```

### No método main:

- Crie um objeto da classe Pessoa usando o construtor

- Passe valores para nome e idade

- Chame o método apresentar()
