# Exercício 2 — Classe Produto

Crie uma classe chamada `Produto`.
A classe deve possuir os seguintes atributos:

- `nome`

- `preco`

- `quantidade`

Crie dois métodos:
`mostrarProduto()`
Deve exibir:
```java
Produto: Notebook
Preço: 3500.0
Quantidade: 5
```

`calcularValorEstoque()`

Esse método deve calcular:

`preco * quantidade`

e exibir o valor total do estoque.

Exemplo de saída
```java
Produto: Notebook
Preço: 3500.0
Quantidade: 5
Valor total em estoque: 17500.0
```

No método main:

1 - Crie um objeto Produto
`
2 - Defina os atributos

3 - Chame os dois métodos
