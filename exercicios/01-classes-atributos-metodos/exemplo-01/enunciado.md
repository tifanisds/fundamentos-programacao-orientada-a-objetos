# Exercício 1 — Classe Carro

Crie uma classe chamada `Carro` que represente um carro.
A classe deve possuir os seguintes atributos:

- `marca`

- `modelo`

- `ano`

Crie também um método:

- exibirInformacoes() → que mostre na tela os dados do carro.

Exemplo de saída esperada

```java
Marca: Toyota
Modelo: Corolla
Ano: 2022
```

## No método main:
1 - Crie um objeto da classe Carro

2 - Defina valores para os atributos

3 - Chame o método exibirInformacoes()
