# 📚 Guia de Programação Orientada a Objetos (POO)

Este repositório tem como objetivo criar um **guia teórico e prático de Programação Orientada a Objetos (POO)**.

O material foi organizado para servir como apoio nos estudos, reunindo **explicações teóricas e exercícios práticos** que ajudam a compreender os principais conceitos da POO.

A linguagem utilizada nos exercícios deste repositório é **Java**.

---

## 📂 Organização do Repositório

O repositório está dividido em duas seções principais:

* **Apostilas**
* **Exercícios**

### 📖 Apostilas

Nas apostilas você encontrará o **conteúdo teórico** sobre os principais conceitos da Programação Orientada a Objetos.

Os temas são explicados utilizando:

* linguagem clara e objetiva
* analogias para facilitar o entendimento
* exemplos práticos de código

O objetivo é tornar os conceitos mais fáceis de compreender, mesmo para quem está tendo o primeiro contato com POO.

---

### 💻 Exercícios

Na seção de exercícios você encontrará **atividades práticas separadas por tópico**.

Cada exercício possui:

* **Enunciado**, descrevendo o problema a ser resolvido
* **Resolução**, com uma possível implementação em **Java**

Os exercícios foram organizados de acordo com os mesmos temas abordados nas apostilas, permitindo praticar cada conceito estudado.

---

OBS: Este repositório está em constante evolução e novos conteúdos e exercícios podem ser adicionados ao longo do tempo.
